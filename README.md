# House Price Prediction

This project predicts house prices using Machine Learning.

## Dataset

The dataset is large, so it is not included in this GitHub repository.

Download the dataset from:
https://drive.google.com/file/d/19y8kQgKeYgXLcWNXv5Yrc34AqnJDlXVp/view?usp=drivesdk

After downloading:
1. Create a folder named `data`
2. Place the CSV file inside the `data` folder

Example:
data/house_Price_Prediction.csv

## How to Run

Open and run:
house_price_prediction.ipynb

The notebook uses this path:
pd.read_csv("data/house_Price_Prediction.csv")
